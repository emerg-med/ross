apTabMenu Example #5

This example demonstrates:

  • Tab type 0 - "3D tabs" with Windows-like gray design
  • Default selected tab is "Example #5" (pressedItem=6)
  • Tab icon -
  • Disabled "Example #4" tab - target parameter equals "_"
  • Tabs bottom line
  • Browser's status bar shows tab link

HTML Source

<applet Code=apTabMenu Archive=apTabMenu.jar Width=570 Height=22>
<param name="Copyright" value="Apycom Software - www.apycom.com">
<param name="bottomLine" value="true">
<param name="pressedItem" value="6">
<param name="tabType" value="0"> 
<param name="backColor" value="d0d0d0">
<param name="tabColor" value="c0c0c0">
<param name="tabHighColor" value="d0d0d0">
<param name="fontColor" value="000000">
<param name="fontHighColor" value="000000">
<param name="font" value="Arial,11,0">
<param name="menuItems" value="
      {Apycom Home,testlink.html,_self,pix/home.gif}
      {Example #1,ex1.html,_self}
      {Example #2,ex2.html,_self}
      {Example #3,ex3.html,_self}
      {Example #4,ex4.html,_}
      {Example #5,ex5.html,_self}
      {Example #6,ex6.html,_self}
      ">
</applet>
