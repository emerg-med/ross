apTabMenu Example #6

   This example demonstrates:

  • Tab type 4 - "border tabs with shadow"
  • 3D text effects (normal and mouseover text shadow)
  • Default selected tab is "Example #6" (pressedItem=6)
  • No tabs bottom line

   HTML Source

<applet Code=apTabMenu Archive=apTabMenu.jar Width=530 Height=25>
<param name="Copyright" value="Apycom Software - www.apycom.com">
<param name="bottomLine" value="false">
<param name="pressedItem" value="6">
<param name="tabType" value="4"> 
<param name="backColor" value="ccff99">
<param name="tabColor" value="eeffdd">
<param name="tabHighColor" value="77aa88">
<param name="borderColor" value="336644">
<param name="fontColor" value="444422">
<param name="fontHighColor" value="fffff0">
<param name="shadowColor" value="e0e0e0"> 
<param name="shadowHighColor" value="222266"> 
<param name="font" value="Arial,12,1">
<param name="menuItems" value="
            {Example #1,ex1.html,_self}
            {Example #2,ex2.html,_self}
            {Example #3,ex3.html,_self}
            {Example #4,ex4.html,_self}
            {Example #5,ex5.html,_self}
            {Example #6,ex6.html,_self}
            ">
</applet>
