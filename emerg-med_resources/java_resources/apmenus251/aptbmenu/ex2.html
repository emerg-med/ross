apTabMenu Example #2

This example demonstrates:

  • Tab type 0 - "3D tabs"
  • Default selected tab is "Example #2" (pressedItem=2)
  • 3D text effects (normal and mouseover text shadow)
  • Tabs bottom line
  • Browser's status bar shows tab text

HTML Source

<applet Code=apTabMenu Archive=apTabMenu.jar Width=500 Height=24>
<param name="Copyright" value="Apycom Software - www.apycom.com">
<param name="bottomLine" value="true">
<param name="pressedItem" value="2">
<param name="tabType" value="0"> 
<param name="status" value="text">
<param name="backColor" value="e0e000">
<param name="tabColor" value="50d050">
<param name="tabHighColor" value="e0e000">
<param name="fontColor" value="ffffff">
<param name="fontHighColor" value="000000">
<param name="shadowColor" value="404040"> 
<param name="shadowHighColor" value="ffffff"> 
<param name="font" value="Arial,11,1">
<param name="menuItems" value="
    {Example #1,ex1.html,_self}
    {Example #2,ex2.html,_self}
    {Example #3,ex3.html,_self}
    {Example #4,ex4.html,_self}
    {Example #5,ex5.html,_self}
">
</applet>
