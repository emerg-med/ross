apImageMenu Example #1

This example demonstrates:

  • Tiled image background
  • 3-state item images -   
  • Items w/o text - text parameter equals "_"
  • Browser's status bar shows item link
  • Buttons are aligned horizontally

HTML Source

<applet code="apImageMenu" Archive="apImageMenu.jar" Width = "280" Height = "25">
<param name="Copyright" value="Apycom Software - www.apycom.com">
<param name="isHorizontal" value="true">
<param name="backPic" value="pix/backline.gif">
<param name="status" value="link">
<param name="menuItems" value="
            {_,testlink.html,_self,pix/apycom1.gif,pix/apycom2.gif,pix/apycom3.gif}
            {_,testlink.html,_self,pix/apmenus1.gif,pix/apmenus2.gif,pix/apmenus3.gif}
            {_,testlink.html,_self,pix/dnl1.gif,pix/dnl2.gif,pix/dnl3.gif}">
</applet>
