apImageMenu Example #3

This example demonstrates:

  • Default 3-state images for all items -
      
  • User defined browser's status string
  • Centered text
  • Vertical menu direction

HTML Source

<applet code="apImageMenu" Archive="apImageMenu.jar" Width = "144" Height = "68">
<param name="Copyright" value="Apycom Software - www.apycom.com">
<param name="alignText" value="center">
<param name="backColor" value="ffffff">
<param name="backHighColor" value="ffffff">
<param name="fontColor" value="0">
<param name="fontHighColor" value="FFFFFF">
<param name="overSound" value="pix/put.au">
<param name="clickSound" value="pix/pop.au">
<param name="defaultImage" value="pix/bt2.gif,pix/bt1.gif,pix/bt3.gif">
<param name="statusString" value="Select any item!">
<param name="font" value="TimesRoman,11,0">
<param name="menuItems" value="
    {Apycom Home,testlink.html,_self}
    {Apycom Applets,testlink.html,_self}
    {apImageMenu,testlink.html,_self}
    {Download,testlink.html,_self}
    ">
</applet>
