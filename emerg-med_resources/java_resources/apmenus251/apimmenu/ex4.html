apImageMenu Example #4

This example demonstrates:

  • Image background
  • Default image for mouseover state -
  • Defined mouseover state images for "Home", "Examples" and "Contact Us" items -
      
  • Click state images are not defined - applet uses mouseover images for click state
  • Normal and mouseover text shadow
  • Vertical menu direction
  • Text aligned to left

HTML Source

<applet code="apImageMenu" Archive="apImageMenu.jar" Width="130" Height="135">
<param name="Copyright" value="Apycom Software - www.apycom.com">
<param name="alignText" value="left">
<param name="backPic" value="pix/back70.gif">
<param name="defaultImage" value="_,pix/itemsel.gif">
<param name="backColor" value="ffffff">
<param name="backHighColor" value="ffffff"> 
<param name="fontColor" value="000000">
<param name="fontHighColor" value="ffffff">
<param name="shadowColor" value="d0d0d0"> 
<param name="shadowHighColor" value="0077ff"> 
<param name="status" value="link">
<param name="font" value="Arial,11,1">
<param name="menuItems" value="
{ Home,testlink.html,_self,_,pix/item11.gif}
{ Examples,testlink.html,_self,_,pix/item21.gif}
{ Parameters,testlink.html,_self} 
{ FAQ,testlink.html,_self}
{ Download,testlink.html,_self}
{ Contact Us,mailto:support@apycom.com,_self,_,pix/item31.gif}">
</applet>
