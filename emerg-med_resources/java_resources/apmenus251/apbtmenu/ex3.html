apButtonMenu Example #3

This example demonstrates:

  • Applet's 3D border
  • Item icons -
  • Disabled items - target parameter equals "_"
  • Button type 0 ("no buttons")
  • Separators
  • Vertical menu direction
  • Text aligned to left
  • Browser's status bar shows item link

HTML Source

<applet Code="apButtonMenu" Archive="apButtonMenu.jar" Width = "180" Height = "200">
<param name="Copyright" value="Apycom Software - www.apycom.com">
<param name="isHorizontal" value="false">
<param name="3DBorder" value="true">
<param name="buttonType" value="0"> 
<param name="status" value="link">
<param name="alignText" value="left"> 
<param name="backHighColor" value="336699">
<param name="backColor" value="d0e0b0">
<param name="fontHighColor" value="fffff0">
<param name="fontColor" value="000000">
<param name="font" value="Arial,11,0">
<param name="menuItems" value="
     {Apycom Software,http://www.apycom.com,_self,pix/home32.gif}
     {-}
     {Menu Applets,testlink.html,_self,pix/progs32.gif}
     {Parameters,testlink.html,_,pix/setup32.gif}
     {Examples,testlink.html,_,pix/help32.gif}
     {Download Applets,http://www.apycom.com/download/apmenu.zip,_self,pix/run32.gif}
     {-} 
     {Contact Us,mailto:support@apycom.com,_self,pix/mail32.gif}
     ">
</applet>
