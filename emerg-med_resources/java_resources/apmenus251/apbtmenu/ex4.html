apPopupMenu Example #4

This example demonstrates:

  • Javascript calls. For more info see "Javascript tips"
  • Item icons -
  • Applet's 3D border
  • Button type 0 ("no buttons")
  • Separators
  • Vertical menu direction
  • Text aligned to left

HTML Source

<applet Code=apButtonMenu Archive=apButtonMenu.jar Width=245 Height=160 MAYSCRIPT>
<param name="Copyright" value="Apycom Software - www.apycom.com">
<param name="isHorizontal" value="false">
<param name="3DBorder" value="true">
<param name="buttonType" value="0">
<param name="status" value="link">
<param name="alignText" value="left">
<param name="backColor" value="aaccaa">
<param name="backHighColor" value="ff6600">
<param name="fontColor" value="000000">
<param name="fontHighColor" value="ffffff">
<param name="font" value="Arial,11,0">
<param name="menuItems" value="
     {Apycom,testlink.html,_self,pix/new-41.gif}
     {-}
     {Click to call message box,javascript:1,_,pix/new-31.gif}
     {Click to call confirmation dialog,javascript:2,_,pix/new-31.gif}
     {Open 'Javascript Tips' in 680x600 window,javascript:3,_,pix/new-31.gif}
     {Open 'Javascript Tips' in 300x200 window,javascript:4,_,pix/new-31.gif}
     {Launch javascript function,javascript:5,_,pix/new-31.gif}
     {-}
     {Contact,mailto:support@apycom.com,_self,pix/new-51.gif}
     ">
<param name="javascript:1" value="alert('Hello world!')">
<param name="javascript:2" value="confirm('Do you want to confirm?')">
<param name="javascript:3" value="open('javascript.html','_blank','width=680,height=600')">
<param name="javascript:4" value="open('javascript.html','_blank','width=300,height=200')">
<param name="javascript:5" value="my_js_function()">
</applet>
