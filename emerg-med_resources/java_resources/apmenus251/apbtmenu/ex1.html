apButtonMenu Example #1

This example demonstrates:

  • Image background
  • Item icons -    
  • Button type 3 ("mouseover fat button")
  • Browser's status bar shows item text
  • Buttons are aligned horizontally

HTML Source

<applet Code="apButtonMenu" Archive="apButtonMenu.jar" Width="480" Height="23">
<param name="Copyright" value="Apycom Software - www.apycom.com">
<param name="isHorizontal" value="true">
<param name="status" value="text">
<param name="buttonType" value="3">
<param name="backPic" value="pix/blueblur.jpg">
<param name="backColor" value="ffffff">
<param name="fontColor" value="ffffff">
<param name="font" value="Arial,11,1">
<param name="menuItems" value="
    {Apycom,testlink.html,_self,pix/ghome.gif}
    {apMenus,testlink.html,_self,pix/gopen.gif}
    {Parameters,testlink.html,_self,pix/gtable.gif}
    {Download,testlink.html,_self,pix/gsave.gif}
    {Links,testlink.html,_self,pix/glink.gif}
">
</applet>
