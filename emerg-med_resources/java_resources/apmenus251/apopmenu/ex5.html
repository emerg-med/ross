apPopupMenu Example #5

This example demonstrates:

  • Mouse over sound and click sound
  • Javascript calls. For more info see "Javascript tips"
  • Item icons -   
  • 3D arrows
  • Font for submenus is the same as for top menu
  • Disabled items - see "Downloads"
  • Button type 1 ("flat buttons")
  • Separators
  • Horizontal menu direction
