apPopupMenu Example #1

This example demonstrates:

  • Tiled image background -
  • Item icons -
  • 3D arrows
  • System font for submenus
  • Button type 3 ("mouseover fat button")
  • Subitem separators
  • Buttons are aligned horizontally

HTML Source
