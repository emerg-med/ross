apPopupMenu Example #4

This example demonstrates:

  • Javascript calls. For more info see "Javascript tips"
  • Item icons -
  • Applet's 3D border
  • Solid arrows
  • Button type 0 ("no buttons")
  • Separators
  • Vertical menu direction

HTML Source

<applet Code=apPopupMenu Archive=apPopupMenu.jar Width=175 Height=180 MAYSCRIPT>
<param name="Copyright" value="Apycom Software - www.apycom.com">
<param name="isHorizontal" value="false">
<param name="3DBorder" value="true">
<param name="solidArrows" value="true">
<param name="buttonType" value="0"> 
<param name="status" value="link">
<param name="alignText" value="left"> 
<param name="backColor" value="b5c3d6">
<param name="backHighColor" value="4a6184">
<param name="fontColor" value="000000">
<param name="fontHighColor" value="FFFFFF">
<param name="font" value="Arial,11,0">
<param name="menuItems" value="
      {Apycom,pix/bhome.gif}
      {|Home,http://www.apycom.com,_blank}
      {|-}
      {|apMenu Home,testlink.html,_self}
      {|apPopupMenu Home,testlink.html,_self}
      {|apButtonMenu Home,testlink.html,_self}
      {|apImageMenu Home,testlink.html,_self}
      {|-}
      {|Contact,testlink.html,_self}
      {-}
      {Javascript tests,pix/bfolderc.gif}
      {|Click to call message box,javascript:1,_}
      {|Click to call confirmation dialog,javascript:2,_}
      {|Open 'Javascript Tips' page in a 680x600 window,javascript:3,_}
      {|Open 'Javascript Tips' page in a 300x200 window,javascript:4,_}
      {|Launch javascript function,javascript:5,_}
      {-}
      {Call message box,javascript:1,_,pix/bbook.gif}
      {Call confirmation dialog,javascript:2,_,pix/bbook.gif}
      {Open 680x600 window,javascript:3,_,pix/bbook.gif}
      {Open 300x200 window,javascript:4,_,pix/bbook.gif}
      {Launch javascript function,javascript:5,_,pix/bbook.gif}
      {-} 
      {Contact,mailto:support@apycom.com,_self,pix/bmail.gif}
      ">
<param name="javascript:1" value="alert('Hello world!')">
<param name="javascript:2" value="confirm('Do you want to confirm?')">
<param name="javascript:3"
            value="open('javascript.html','_blank','scrollbars,width=680,height=600')">
<param name="javascript:4"
            value="open('javascript.html','_balnk','scrollbars,width=300,height=200')">
<param name="javascript:5" value="my_js_function()">
</applet>
