Contributing

Grant Taylor

Welcome to the Linux Printing website. This site contains a number of resources to help you with your Linux Printing problems, including the Printing HOWTO, a support database, a project list, printer vendor scorecards, Foomatic, and more.

Researching a printer purchase? Start with suggested printers, or browse the database.

Looking for help? If the HOWTO and database don't help you, try asking a human in the forums.

Daniel Gordini has put out the first stable release of his Lexmark 3200 driver. This also drives the Compaq IJ750, which is really the same printer. See Daniel's website.

Ghostscript 6.50 out!Wed Dec 6 03:08:28 2000
Raph has announced the release of Ghostscript 6.50. This includes assorted bug fixes and significant PDF "distiller" improvements; it also means that 6.01 will shortly be under the GPL and thus more widely available.

Discuss in linuxprinting.general.

Bruce Perens hired by HPWed Dec 6 02:58:29 2000
It's only partly printing-related, but it's perhaps a sign of better things to come: Bruce Perens, longtime free software advocate, has been hired as a sort of corporate free software guru at HP. The ensuing Slashdot threads are mostly just rants around HP's lame developer support for printers; few new facts were forthcoming, although rumours of impending progress have certainly found new fuel.

More ImprovementsSun Dec 3 23:42:13 2000
I've made a bunch of improvements; the most obvious user-visible one is the new printer summary list format, which now uses a table instead of the rather hard-to-read description list format. I also fixed assorted buglets throughout the data and foomatic backend scripts. The edit printer form, in particular, has been greatly improved.
Foomatic library improvements; magicfilter supportSun Dec 3 00:12:36 2000
I've made substantial improvements to the Foomatic library; now you just have to download the latest snapshot, type make install, and it will snarf the database over, etc.

I've also put out the first beta of my magicfilter with Foomatic support. It is intended for use with LPRng, where it should offer reasonably simple installation, the usual extensive Foomatic driver support, etc.

Also, the online database (and offline tools) now support multiple branches in the data. So there can be "snapshots" to reflect particular driver or distribution releases. Those snapshots can be modified live. The only thing missing is merge tools...

Finally, the code has been mostly migrated into CVS.

Ghostscript Security Bug!Mon Nov 27 11:00:43 2000
Ghostscript has been found to have a tempfile bug (which I assume exists everywhere) and, on Debian at least, a build-time environment bug, both of which could lead to various sorts of local attacks. Debian's announcement. Everyone should update to a Ghostscript with this problem fixed.
